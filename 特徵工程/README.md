# 特徵工程 (Feature Engineering)

機器學習中一個至關重要的步驟：特徵工程。良好的特徵工程可以顯著提升模型的性能，而差的特徵工程則可能導致模型無法學習到有用的資訊。

## 什麼是特徵工程？

特徵工程是利用領域知識來建立特徵，讓機器學習演算法得以運作的過程。它包含了特徵的創造、轉換、提取和選擇。
